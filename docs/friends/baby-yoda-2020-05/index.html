'The Child'

“The Child” automaton version 1, drinking soup. A lever under the base allows Baby Yoda’s head to rotate side to side.

Movement Video: Head turning

Material: Hand carved, sanded, and acrylic painted basswood figure; Bamboo, walnut base and cherry kinetic components sealed with Odie’s Oil.

Approximate dimensions (inches): Figure only - 3” H x 2.5” W x 2” D, with base - 4.25” H x 2.5” W x 3.5” D

Completion Date: May 4, 2020
